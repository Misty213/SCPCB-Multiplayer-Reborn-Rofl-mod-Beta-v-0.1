# Как установить мод на игру?
Для того чтобы запустить мод вам нужна выполнить шаги ниже :
1) Скачайте SCP: CB Multiplayer Reborn из стима (https://store.steampowered.com/app/3257000/SCP_CB_Multiplayer_Reborn/)
2) Скачайте сам мод (репозиторий)
3)После скачивание игры и мода зайдите в стим во вкладку "библиотека"
4)Дальше в библиотеке нажмите на вкладку игры
5)Потом на иконку шестеренки и нажмите кнопку "Свойства"
6) в открывшимся окне  передите на вкладку "Установленные файлы" нажимаете на кнопку "Обзор"
7) Переносите все с файла мода в окно которые открылось
8)В момент переноса откроется окно там 3 пункта выбераете пунктв котором имеется слова "Заменить"
9)Когда мод перенося все мод установлен на игру
